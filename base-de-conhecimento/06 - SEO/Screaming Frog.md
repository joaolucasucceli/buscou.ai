# Screaming Frog SEO Spider

## O que e

O Screaming Frog SEO Spider e um crawler de sites (rastreador) para desktop que analisa URLs e coleta dados tecnicos essenciais para [[SEO Tecnico]]. E considerado a ferramenta padrao da industria para auditorias tecnicas de SEO, capaz de identificar mais de 300 problemas, alertas e oportunidades de melhoria em SEO, saude do site e experiencia do usuario.

## Principais Funcionalidades

### Auditoria de Links
Rastreia o site instantaneamente e encontra links quebrados (erros 404), erros de servidor (5xx), links internos e externos com problemas.

### Analise de Redirecionamentos
Identifica redirecionamentos temporarios e permanentes, cadeias de redirecionamento e loops. Permite tambem carregar uma lista de URLs para auditar migracoes de site.

### Analise de Metadata
Analisa titulos de pagina e meta descriptions durante o rastreamento, identificando aqueles que sao muito longos, muito curtos, ausentes ou duplicados.

### Conteudo Duplicado
Descobre URLs duplicadas usando verificacao algoritmica MD5, elementos parcialmente duplicados (titulos, descricoes, headings) e paginas com pouco conteudo.

### Dados Estruturados (Schema)
Valida markup de dados estruturados (Schema.org) e identifica erros de implementacao.

### Sitemaps XML
Gera sitemaps XML automaticamente e identifica problemas nos sitemaps existentes.

### Renderizacao JavaScript
Rastreia sites construidos com JavaScript (React, Angular, Vue) renderizando as paginas como o Googlebot faria.

### Integracao com IA (Novidade 2026)
A partir da versao 21.0, o Screaming Frog integrou APIs de IA diretamente na ferramenta. Na versao 22.0, foi expandido significativamente com suporte para OpenAI, Gemini, Anthropic (Claude) e Ollama via Config > API Access > AI. Permite usar IA para analisar conteudo, gerar meta descriptions e classificar paginas automaticamente.

### Integracoes com APIs
Conecta-se ao [[Google Analytics]], [[Google Search Console]], PageSpeed Insights e Ahrefs/Majestic para enriquecer os dados do rastreamento.

## Como Usar para SEO Tecnico

1. **Baixe e instale** a ferramenta (disponivel para Windows, Mac e Linux)
2. **Insira a URL** do site que deseja rastrear
3. **Configure filtros** conforme necessario (excluir subdominios, limitar profundidade)
4. **Analise os resultados** nas abas: Internal, External, Protocol, Response Codes, etc.
5. **Exporte dados** para planilhas ou gere relatorios automaticos
6. **Priorize correcoes** com base na severidade e impacto
