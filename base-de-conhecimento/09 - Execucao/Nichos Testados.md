## Ranking: Nichos Mais Lucrativos

Ordene seus nichos por lucratividade para priorizar esforcos de prospecao. Atualize conforme acumular dados.

| Ranking | Nicho | Ticket Medio | Margem | Facilidade de Resultado | Nota Geral |
|---|---|---|---|---|---|
| 1 | _SaaS B2B_ | _R$ 5.000_ | _Alta_ | _Media_ | _9/10_ |
| 2 | _E-commerce_ | _R$ 3.500_ | _Media_ | _Media_ | _7/10_ |
| 3 | _Clinicas saude_ | _R$ 2.000_ | _Media_ | _Alta_ | _7/10_ |
| | | | | | |
| | | | | | |

**Criterios para avaliar lucratividade**:
- Ticket medio: quanto o cliente paga por mes
- Margem: custo de entrega vs receita (considere horas, ferramentas, subcontratacao)
- Facilidade de resultado: quao rapido voce consegue mostrar ROI ao cliente
- Retencao: quanto tempo o cliente permanece ativo (nichos com resultados visiveis retêm mais)
- Escalabilidade: voce consegue atender multiplos clientes no mesmo nicho sem conflito?

## Padroes Observados entre Nichos

Conforme voce acumular experiencia em 3+ nichos, documente padroes aqui:

- **Quais estrategias funcionam em todos os nichos?** (ex: [[Content Clustering e Pillar Pages]], [[Framework AIO Completo]])
- **Quais sao nicho-especificas?** (ex: [[SEO para E-commerce]] so para lojas, [[Local SEO]] so para negocio fisico)
- **Qual o tempo medio ate resultado por tipo de nicho?**
- **Quais nichos tem mais oportunidade de AIO/GEO em 2026?** Consulte [[Como IA Escolhe Respostas]] para entender os fatores

## Como Usar Esta Nota

1. **Antes de prospectar**: Consulte nichos testados para alinhar discurso de venda
2. **Em calls de venda**: Cite resultados de nichos similares ao do prospect. Use dados do [[Dashboard de Resultados]]
3. **Na [[Proposta Comercial - Modelo]]**: Inclua case de nicho similar como prova
4. **No [[Relatorio para Cliente - Modelo]]**: Compare resultados do cliente com benchmarks do nicho
5. **Para decisoes estrategicas**: Use o ranking de lucratividade para escolher onde investir tempo
