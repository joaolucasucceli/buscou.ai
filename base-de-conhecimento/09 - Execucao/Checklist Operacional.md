# Checklist Operacional

> Rotina operacional para manter o pipeline funcionando. Use como SOP (Standard Operating Procedure) para voce ou sua equipe.

---

## Diario (15-30 min)

- [ ] Verificar [[Google Search Console]] (erros, cobertura)
- [ ] Checar rankings das keywords prioritarias
- [ ] Responder comentarios e interacoes
- [ ] Anotar ideias de conteudo que surgirem

## Semanal (2-4 horas)

### Segunda - Planejamento
- [ ] Revisar metricas da semana anterior
- [ ] Definir prioridades da semana (qual etapa do [[00 - Pipeline|Pipeline]])
- [ ] Atualizar [[Dashboard de Resultados]]

### Terca/Quarta - Producao
- [ ] Criar/otimizar 2-3 artigos
- [ ] Implementar schema markup pendente
- [ ] Fazer interlinking dos novos conteudos

### Quinta - Distribuicao
- [ ] Publicar 2-3 posts em plataformas externas (LinkedIn, Reddit, Medium)
- [ ] Fazer outreach para 3-5 oportunidades de link building
- [ ] Seguir plano de [[11 - Distribuicao/]]

### Sexta - Teste e Documentacao
- [ ] Testar 5-10 prompts em IAs (ChatGPT, Perplexity)
- [ ] Documentar resultados em [[Queries que Rankeamos]]
- [ ] Registrar experimentos no [[Template - Experimento AIO]]
- [ ] Atualizar [[Palavras-Chave Dominadas]]

## Mensal (4-6 horas)

- [ ] Auditoria tecnica rapida ([[Playbook - Auditoria Relampago]])
- [ ] Relatorio de performance ([[Relatorio para Cliente - Modelo]])
- [ ] Revisar e atualizar conteudo antigo (content refresh)
- [ ] Analisar concorrentes (quem subiu, quem caiu)
- [ ] Retestar todos os prompts baseline de IA
- [ ] Atualizar [[Nichos Testados]] e [[Dashboard de Resultados]]
- [ ] Planejar conteudo do proximo mes

## Trimestral (1 dia)

- [ ] Revisao estrategica completa
- [ ] Auditoria SEO + AIO profunda ([[Auditoria SEO - Checklist]])
- [ ] Ajustar estrategia de clusters e keywords
- [ ] Revisar [[ICP - Cliente Ideal]] e posicionamento
- [ ] Atualizar [[Precificacao de Servicos SEO]]
- [ ] Verificar [[Tendencias Globais]] e adaptar

---
