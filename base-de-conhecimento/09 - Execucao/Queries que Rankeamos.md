# Queries que Rankeamos em IA

## Objetivo

Documentar em quais queries de inteligencia artificial seu conteudo (ou do cliente) e citado, mencionado ou recomendado. Enquanto [[Palavras-Chave Dominadas]] rastreia rankings em busca tradicional (Google), esta nota rastreia a **visibilidade em motores de IA** — o novo campo de batalha de 2026. Os dados aqui alimentam o [[Dashboard de Resultados]] e informam a estrategia de [[Framework AIO Completo|AIO]].

## Prompts que Mais Convertem

Nem toda query de IA tem o mesmo valor. Identifique quais queries trazem mais impacto real:

| Query | Plataforma | Tipo de Intencao | Volume Estimado | Valor para o Negocio | Cliente |
|---|---|---|---|---|---|
| _"melhor CRM para startups"_ | _ChatGPT_ | _Comercial_ | _Alto_ | _Gera leads diretos_ | _Cliente A_ |
| _"dentista emergencia zona sul"_ | _Google AIO_ | _Transacional_ | _Medio_ | _Agendamentos imediatos_ | _Cliente C_ |
| _"o que e SEO"_ | _Perplexity_ | _Informacional_ | _Alto_ | _Branding, autoridade_ | _Proprio_ |
| | | | | | |

**Classificacao de intencao para queries de IA**:
- **Informacional**: Usuario quer aprender. Valor = branding e autoridade
- **Comercial**: Usuario esta pesquisando opcoes. Valor = leads e consideracao
- **Transacional**: Usuario quer agir agora. Valor = conversao direta
- **Navegacional**: Usuario busca marca especifica. Valor = protecao de marca

Priorize queries comerciais e transacionais para clientes. Para seu proprio negocio, queries informacionais constroem autoridade que alimenta o [[E-E-A-T]].

| Analise de tendencias | Mensal | Esta nota |
| Revisao de estrategia | Trimestral | [[Framework AIO Completo]] |

### Como Testar Queries Manualmente

1. Abra ChatGPT, Perplexity, Gemini e Google (para AI Overviews)
2. Digite a query exatamente como um usuario faria
3. Verifique se seu conteudo/marca e citado
4. Registre: posicao na resposta, URL citada, tom da mencao
5. Anote quais concorrentes sao citados (gap analysis)
6. Repita em 2-3 dias diferentes para confirmar consistencia

### Quando uma Query Nao e Citada — Plano de Acao

Consulte [[Como IA Escolhe Respostas]] e aplique:
1. Verifique se crawlers de IA tem acesso (robots.txt) — [[llms.txt e Acessibilidade para Crawlers IA]]
2. Reescreva o conteudo com formato answer-first
3. Adicione dados e estatisticas com fontes
4. Publique o conteudo em 3+ plataformas externas
5. Implemente [[Schema Markup para IA]] na pagina
6. Aguarde 30-90 dias e reteste
