## Como Usar Este Dashboard

### 1. Antes de Calls de Venda
- Consulte as "Frases de Impacto" acima
- Verifique se tem case no nicho do prospect em [[Nichos Testados]]
- Prepare 2-3 dados especificos para citar na conversa
- Tenha pronto o ROI medio e tempo medio ate resultado

### 2. Em Propostas Comerciais
- Inclua a tabela "Metricas por Nicho" filtrada para o nicho do prospect
- Adicione 1-2 cases detalhados dos "Top 5 Melhores Resultados"
- Use o template da [[Proposta Comercial - Modelo]] com dados reais deste dashboard
- Diferencie-se mostrando resultados de AIO (citacoes em IA) — poucos concorrentes oferecem isso

### 3. Em Relatorios para Clientes
- No [[Relatorio para Cliente - Modelo]], compare resultados do cliente com benchmarks do dashboard
- Mostre evolucao mensal especifica do cliente
- Contextualize: "Seu crescimento de XX% esta acima/na media dos nossos clientes no nicho"

### 4. Para Conteudo e Marketing Proprio
- Use dados agregados em posts no LinkedIn
- Cite resultados em artigos do blog (sem revelar dados confidenciais do cliente)
- Construa [[Prova Social]] com depoimentos e metricas reais
- Dados proprios aumentam sua visibilidade em IA — veja [[Como IA Escolhe Respostas]]

### 5. Para Decisoes Estrategicas
- Identifique nichos com melhor ROI para priorizar prospecao
- Avalie quais servicos geram mais resultado (SEO puro vs SEO + AIO)
- Defina metas realistas baseadas em historico real
- Ajuste [[Precificacao de Servicos SEO]] com base em resultados comprovados
