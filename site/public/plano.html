Auth + Painel Admin

Sem painel pra gerenciar clientes, cupons e financeiro, nada começa.

  • Rota /login com Supabase Auth e redirect role-based
  • Painel /admin com visão geral + CRUD de clientes

  • Painel /admin/financeiro — MRR, lista compras, Payment Links emitidos

Pagamento + onboarding cliente

Cliente paga → vira organização → entra no painel dele.

  • Schema: compras, assinaturas_infra, parcelas, clientes_stripe
