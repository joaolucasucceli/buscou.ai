# /agentes/contexto

Contexto compartilhado que **todo agente** recebe como system prompt.

Arquivos:
- `verdade-unica.md` — snapshot da VERDADE_UNICA_BUSCOU (com data de sincronizacao).
- `icp.md` — resumo do ICP por nicho (para o agente saber para quem escrever).
- `oferta.md` — resumo da oferta (R$ 2.500 a vista / R$ 3.000 em 12x, sem mensalidade).
- `tom-de-voz.md` — regras de voz (direto, com numeros, estrutura problema→mudanca→prova).
- `linguagem.md` — tabela de termos proibidos vs permitidos.

**Atualizacao:** quando VERDADE_UNICA for alterada (Decision Log), atualizar os arquivos aqui e bumpar versao.
