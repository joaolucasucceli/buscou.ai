# /conteudo/clusters

Planos de topic clusters por nicho/cliente.

Cada cluster: 1 pillar page + N cluster pages interligados, cobrindo intencao principal + subtopics.

Nomenclatura: `{nicho}_{tema}.md` (ex.: `clinicas_harmonizacao-facial.md`).

Referencia: [[Content Clustering e Pillar Pages]] em [08 - Estrategia Conteudo/](../../base-de-conhecimento/08 - Estrategia Conteudo/).
