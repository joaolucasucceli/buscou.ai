# /conteudo/templates

Templates de artigo por tipo de query.

Tipos cobertos:
- `melhor-x.md` — listicles com ItemList + Product schema
- `vale-a-pena-x.md` — analise + FAQ schema
- `x-vs-y.md` — comparativo com ComparisonTable schema
- `como-fazer-x.md` — HowTo schema + passos numerados
- `local-queries.md` — LocalBusiness + NAP schema
- `informacional.md` — guia denso + FAQPage

Cada template tem: estrutura de headings + skeleton de schema JSON-LD + checklist de answer-first.

Referencia: [08 - Estrategia Conteudo/](../../base-de-conhecimento/08 - Estrategia Conteudo/).
