# /produto/frontend

App Next.js. Contem:
- Landing page de vendas (hero, oferta, checkout).
- Dashboard do cliente (IVT, artigos publicados, status do motor).
- Onboarding (formulario + configuracao do blog).

Stack: Next.js 14 App Router + Tailwind consumindo tokens de [../../identidade-visual/](../../identidade-visual/).
