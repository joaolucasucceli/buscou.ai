# /produto/cms

Servico que hospeda N blogs de clientes. Multi-tenant com subdominios ou dominios customizados.

Responsabilidades:
- Renderizar blog do cliente (Next.js app ou template estatico).
- Aplicar identidade do cliente (logo, cores, tipografia).
- Expor schema JSON-LD, sitemap, llms.txt.
- Otimizar Core Web Vitals.
- Integrar com dominio proprio do cliente.
